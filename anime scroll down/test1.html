Un autre type d'animation

avec des keyframes et animation-timeline

Déclaration CSS

Déclarer des variables

Tout d'abord on construit des déclarations

Cela vas nous permettre d'avoir des repères stables

CSS

copier le code

@property --clip-vertical {

syntax"<percentage>";

initial-value95%;

inheritsfalse;

}
@property --clip-horizontal {

syntax" <percentage> ";

initial-value100%;

inheritsfalse;

}

Le Texte

une div de texte

Ensuite on construit une div contenant notre texte

CSS

copier le code

.stop {

--clip-horizontal:100%;

--clip-vertical:93%;

position:relative;

min-height:70vh;

padding-top:15vh;

width:calc(var(--timeline-width));

padding-inline:50px;

}

Le très

::after

Le très qui apparait est un after

On le construit pour créer les très pointillés qui apparaisse durant le scroll

CSS

copier le code

.stop::after {

position: absolute;

top: 20px;

right: 0px;

width: 50%;

height: 12%;

border-width: 3px 0 0 3px;

border-style: dashed;

border-color: oklch(20% 0.5 313);

clip-path: inset(0 0 var(--clip-vertical) var(--clip-horizontal));

transform-origin: center;

animation: showLine linear both;

animation-timeline: view(block);

animation-range: cover 0% contain 40%;

content: "";

}

Les keyframes

@keyframes

Pour finir on fait les animations avec des keyframes

CSS

copier le code

@keyframes showLine {

0% {

--clip-horizontal800%;

--clip-vertical75%;

}

60% {

--clip-horizontal0%;

--clip-vertical95%;

}

100% {

--clip-horizontal0%;

--clip-vertical0%;

}

}
@keyframes slideIn {

0% {

translate0 3%;

--clip-vertical100%;

}

100% {

--clip-vertical0%;

}

}
