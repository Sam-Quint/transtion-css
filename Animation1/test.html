About this demo

This page demonstrates an article page that has a full page cover card that transforms to a fixed header as you scroll down.

The Code

                    
                        @keyframes sticky-parallax-header-move-and-size {
	from {
		background-position: 50% 0;
		height: 100vh;
                        font-size: calc(4vw + 1em);
                        }
	to {
		background-position: 50% 100%;
		background-color: #0b1584;
		height: 10vh;
                        font-size: 2em;
                        }
}

#sticky-parallax-header {
	position: fixed;
	top: 0;

	animation: sticky-parallax-header-move-and-size linear forwards;
	animation-timeline: scroll();
	animation-range: 0vh 90vh;
                        }

body {
	padding-top: 100vh;
}

Explanation

The cover starts at a height of 100vh and shrinks to 10vh as you scroll the root scroller. This is done over a scroll distance of 90vh , put in place using the animation-range property.

Note that you can‚Äôt use position: sticky; here though, as resizing the cover‚Äôs height would shrink down the entire height of the document, and therefore affecting the available scroll distance. Instead, the cover is positioned using position: fixed; . To counteract the content jumping up in response, a margin-top of 100vh is added to the body.
